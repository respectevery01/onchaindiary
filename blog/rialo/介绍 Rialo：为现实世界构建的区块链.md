# 介绍 Rialo：为现实世界构建的区块链

比特币创造了数字稀缺性。以太坊带来了可编程信任。随后涌现的快速链，都在竞相追求纸面上的速度。

然而，大多数新用户仍在起点附近放弃。

为什么？因为 Web3 应用仍然显得笨重、缓慢且脱节。用户渴望像优步、Instagram 或 WhatsApp 那样无缝且必不可少的体验。但如今，即使是简单的操作也涉及太多步骤、太多摩擦和太少可靠性。

这不是我们曾向自己承诺的去中心化世界。我们决定重新开始。

**隆重推出 Rialo。**

Rialo 是现实世界的区块链：一个去中心化、高性能且符合人体工程学的编程网络，与现实生活互动。

二十年前，翻盖手机进化了，GPS、摄像头和互联网连接全部集成到一个设备中：智能手机。突然间，新的用户体验和全新类别的应用成为可能。优步、Instagram、WhatsApp。

**Rialo 将这种融合带到了区块链领域。**

## 问题出在哪里

尽管新链拥有出色的 TPS（每秒交易处理量），但采用率仍然低得令人沮丧。原因显而易见：并非所有人都愿意离开舒适的 Web2 去接触 Web3。

当今的区块链如同孤岛，对现实世界的事件视而不见。它们无法直接与经过数十年打磨的丰富现实世界服务生态系统互动。接入 Web3 应用的过程可能显得笨拙且不直观。上链仅限于极少数敢于应对其复杂性的人。

为了触达普通用户，我们坚信 Web3 必须融入现实世界所在的 Web2，让用户在此轻松入门。

Rialo 弥合了这一鸿沟。凭借现实世界的能力，Rialo 最终让开发者能够构建去中心化应用，这些应用感觉像是易用的现代软件，而非摇摇欲坠的纸牌屋。

因为若没有与现实世界的真实连接，整个体验便会土崩瓦解：

#### 用户们，你们一定遇到过这些问题：

- 互操作性困境：一天内多次切换网络，当资产通过不稳定的跨链桥转移时你们屏息凝神，还要应对各种封装代币的字母组合难题。
- 钱包成为瓶颈：所有操作都发生在割裂的钱包生态中，用户体验如同噩梦，需要不断签署交易确认。
- 难以预料的成本：交易费用可能瞬间暴涨，三明治攻击随时可能突袭耗尽余额。当预言机出现故障时，仓位甚至可能被错误清算。
- 放弃你的社交身份：你在 TikTok 和 X 等平台上多年积累的粉丝，为何要在加密世界从头开始？为何不能带着你的社交身份一同迁移？
- 你需要时刻在线：设定凌晨 3 点的 NFT 发售和代币预售闹钟吧，因为加密世界是全球性的，可不会顾及你的美容觉。
- 最昂贵的一张纸：在 Web2 时代，双重认证（2FA）为密码泄露时的账户访问提供了额外保护。而在钱包黑客攻击猖獗且不可逆的 Web3 世界，2FA 早已不是便利功能，而是迟来的必需品。

开发者们，这些问题你们一定深有体会：

- 深陷集成泥潭：你仅有 10%的时间用于编写实际业务逻辑，其余时间都在与第三方集成和脆弱的链下服务缠斗，消耗着你的运营资金。
- 读取之痛：200 毫秒的最终确认延迟？听起来不错。但你的数据却要穿越层层验证者、全节点、索引器和后端服务，耗时数秒甚至更久才能抵达用户。应用频频卡顿，用户面对陈旧数据干瞪眼——系统的速度永远受制于最慢的环节。
- 无休止的监控与脆弱的响应：你部署了专职服务，不知疲倦地监控现实世界或链上状态，并触发精准计时的交易。然而一旦网络拥堵，你的交易便石沉大海，除非支付高昂的小费。
- 昂贵的索引器：你被迫运行昂贵的全节点和索引基础设施，每月向云服务商支付数千费用，却仅仅为了获取实时数据中的冰山一角。
- 预言机税：需要在链上定制数据源？预言机服务每日收费数千。更糟的是，当它们未能完成交易时，你只能被迫承担坏账。
- 连接必要的恶：你被迫在信任模型中引入桥梁，这立刻削弱了第一层提供的强大安全性。每座桥梁都为利用敞开了大门，并削弱了整个系统。但你又有何选择呢？

这些问题是一个更深层次问题的表现。

多年来，区块链开发者一直困于性能表演，痴迷于追逐 TPS 等虚荣指标，却忽视了真正的问题。仅凭更快的链无法修复糟糕的用户体验、开发者的挫败感以及脱节的基础设施。

路径依赖带我们至此，却无法引领我们走得更远。是时候从根源重新思考，因此我们正在构建一个直观、无缝且与现实世界深度连接的分散式可编程网络。

## 真实世界链

Rialo 以解决当今问题为核心而构建。它并非零散组件的拼凑，而是从零开始重新构想的区块链基础设施，体现了去中心化、可编程网络应有的本质：简洁、富有表现力、无缝连接且互联互通。

- 真实世界数据：通过智能合约中的一行 HTTPS 调用，随时随地获取实时数据。
- 现实世界连接：智能合约中无缝集成任何链下 API。一行代码将您的去中心化应用带向世界，也将世界带入您的去中心化应用。
- 现实世界隐私：快速、经济且强大的解决方案，兼顾可验证与机密执行的最佳优势。
- 真实世界身份：使用电子邮件、短信或社交账户作为您的 Web3 通行证。凭借信用评分获取加密货币贷款。通过智能合约遗嘱传承资产。欢迎使用您的身份加入 Rialo。
- 真实世界速度：用户端到端延迟真正低于秒级。为构建者加速市场推广进程。
- 现实世界可扩展性：适用于全球规模的无限扩展能力，具备多并发提议者等众多特性
- 现实世界反应性：无需昂贵的链下监控循环和 Gas 战争，即可实现闪电般的链上反应。交易即时、自动且可靠地响应。
- 现实世界可编程性：欢迎 `Future` / `Promise` / `.await` 、事件驱动逻辑、随机性及其他人体工程学特性融入智能合约。编写能够休眠、循环与恢复的智能合约。完全上链，跨区块运行，由 RISC-V 驱动。
- 现实世界可用性：恢复熟悉的体验，如双重认证、定时交易和社交登录。区块链悄然退居幕后，正如它本该如此。
- 现实世界资产：将主流及长尾资产引入链上。不仅可能，而且毫不费力。
- 现实世界应用：构建超越投机、人们日常使用的应用程序。

Rialo 采用超模块化设计，各项能力相互增强。它们共同构建了一个整体远大于部分之和的系统。强大的协同效应默认使应用程序更简洁、更快速、功能更强大。
