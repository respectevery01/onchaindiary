# 理解机密 ERC-20
深入了解机密代币实现

我们将创建一个机密的 ERC20 合约，该合约支持：
- 铸造新代币
- 在账户间转移代币
- 管理代币授权额度
- 保持所有金额加密状态
- 仅所有者有权查看所有余额
- 可选的全局解密单个余额功能
